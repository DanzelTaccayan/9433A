Java Script Data Types

Dynamic typing

Java script is a dynamic language which means that you don’t have to declare the type of variable. The type will automatically be assigned once the code is processed

Objects (Properties)

In java script, an object can be seen as a collection of properties. Properties are identified by key values. There are two types of object properties. The data property and the accessor property

Data Property – associates a key with a value
