HTML Syntax

Parts of HTML syntax in order:
1. A single U+FEFF BYTE ORDER MARK (BOM) character
2. Any number of comments and space characters
3. A doctype
4. An html element with its attributes and its contents.


The start tags and end tags of the html element and other particular product of html element can be precluded. Even preclusion of those particular elements occurs, the documents are still considered. To contain the elements, there should be start and end tags.
