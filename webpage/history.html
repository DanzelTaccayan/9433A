1969 - DARPA Developed ARPA

ARPA
Packet switching - chopping the data to be sent into smaller pieces and then are sent to the destined recipient where it then assembles the chopped data
Store and forward

1972 - E-mail

Before the World Wide Web :

WAIS
A system that enables client to search and retrieve documents in a central database. Unlike the WWW, WAIS does not link related documents together, instead documents are stored in databases where you have to query the document/s you want to retrieve.
An Example would be a document (Document_1) you are reading is related to another document (Document_2), in order to get Document_2 you have to query that document in order to retrieve it.
Gopher
A protocol that is used for searching and retrieving documents. It uses Indexes to Subindexes in order to link other documents together("Hierarchies")
It was termed Gopher because like the animal Gopher it burrows from document to document until it reaches its destination.
Usenet
It is like a "Bulletin Board" or a "Forum". Its modern correspondence would be StackOverflow, and Reddit.

1989 - WWW was created

Links documents that are related to each other by using hypertext

The motiviation for the creation of the WWW is for the use of researchers and scientist
