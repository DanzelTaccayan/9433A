Introduction


Web System

A web system can also be called application because it is an application that is created and designed to be used on the Web.

Web Technology

Web technology is the foundation and use of components that make it attainable for different devices to share resources and communicate with each other. It is a factor of computer networks: Wide Area Network (WAN), Local Area Network (LAN) or Metropolitan Area Network (MAN), such as the Internet.
