What is Hypertext Markup Language(HTML)?

Uses Markups.
Example: <h1>Hello World</h1> - Marking up the Hello World text

Language used to create a webpage

Used to design a webpage(meaning its content).

Structure & Content: actual information contained in a webpage

Look at a webpage as a document because documents have a structure

HTML should solely be for content

History of HTML

  • HTML 1.0 - first release
  • 5 years after: HTML 2.0 - RFC 1866
  • HTML 3.0
  • HTML 3.2 - was transferred to W3C which superseded HTML 3.0
  • HTML 4.0 - 1997
  • HTML 4.1 - 1999
  • W3C stopped working on HTML and started working on XHTML(Extensible Hypertext Markup Language) 1.0 - 2000
  • XHTML made HTML stricter(because it is case sensitive and endtags are required) and so it was forgotten
  • Ian Hickson and WHATWG worked on HTML 5 - 2004
  • W3C joined in to support HTML 5 - 2014
