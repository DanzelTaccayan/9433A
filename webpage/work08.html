Entity Header Fields

Name Description
Allow This will enumarate all the set of methods that are allowed and supported by the resource indentified by the request-URI.
Content-Encoding This is used as the modifier to a media-type.
Content-Language This is used to describes the natural langauages which is intended to use to audiences for the enclosed entity.
Content-Length This will show the size of the entity-body.
Content-Location This might be used to supply the resource location for the entity enclosed in a message.
Content-MD5 "This is an MD5 digest of an entity-body for the purpose of providing an end-to-end message integrity check (MIC) of the entity-body" as defined in RFC 1864[23].
Content-Range This is sent with a partial entity-body which is used to specify where to put and apply in the full entity-body this partial body.
Content-Type This will show the media type of the entity-body that was sent to a recipient.
Expires This will provide you the date and time after a response is considered stale.
Last Modified This will show the date and time to which the origin server was believed the variant was last modified.
