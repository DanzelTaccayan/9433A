Status Codes

Status Codes

Informational (1xx)
(xxx) Name Description
100 Continue To continue
101 Switching Protocol When a client uses upgrade header
Successful (2xx)
(xxx) Name Description
200 OK Success
201 Created Successfully created
202 Accepted Accepted but not processed yet
203 Nonauthorized Information Some are not from origin
204 No Content Successful but doesn't return any value
205 Reject Context Reject for data entry
206 Partial Content Half of the page are loaded
Redirection (3xx)
(xxx) Name Description
300 Multiple Choices More than 1 version (html,txt)
301 Moved Permanently Resource has moved to another address
302 Found Page Found
303 See Other Redirect to another page
304 Not Modified Redirect to cache
305 Use Proxy Access from proxy
306 Unused Reserved
307 Temporary Direct Not accessible here for a temporary time only
Client Error (4xx)
(xxx) Name Description
400 Bad Request Wrong request
401 Unauthorized Access protected
402 Payment Required Used in commerce
403 Forbidden Not supposed to be accessed
404 Not Found Page not found
405 Method Not Allowed Put in not allowed
406 Not Acceptable Everything server will generate, client will not accept
407 Proxy Authentication Required Authentication from proxy
408 Request Time-Out Certain amount of time
409 Conflict Cannot be done because conflict in state of resource
410 Gone No forwarding address
411 Length Required Required to specify the content length
412 Precondition Failed Cannot be processed due to failed condition
413 Request Entity Too Large Payload cannot handle the size
414 Request URI Too large Request URI is too long
415 Unsupported Media Type Media type is not supported. Ex. jpeg format only
416 Request range Not Satisfiable Entity size not found
417 Expectation Failed
426 Upgrade Required
Server Error (5xx)
(xxx) Name Description
500 Internal Error Invalid option
501 Not Implemented Cannot handle request or wrong spelling
502 Bad Gateway Server is full or overloaded
503 Service Unavailable Unavailable due to maintenance
504 Gateway Time-Out Server is full or overloaded
505 HTTP Version Not Supported Version is outdated or not supported
